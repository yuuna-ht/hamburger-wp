# 架空のハンバーガーショップサイト
Webエンジニアリングスクール RaiseTech の WordPress副業コースの課題で制作したものです。
架空のハンバーガーショップサイト用の WordPressオリジナルテーマを開発しました。

## 特徴
* **メインビジュアルの管理**：管理画面からメインビジュアル画像やタイトルを簡単に編集可能です。
* **メニュー管理**：管理画面からメニュー項目を簡単に編集可能です。
* **アーカイブページ管理**：管理画面からカテゴリー別ページのタイトルや小見出し、コメントを簡単に編集可能です。
* **ブランチメニュー管理**：フロント画面中央部のブランチメニュー「Eat In」、「Take out」のタイトルや画像、コメントを管理画面から簡単に編集可能です。
* **アクセス管理**：フロント画面下部のアクセスのコメントや、マップ（埋め込みコード）を管理画面から編集可能です。
* **ページング機能**：プラグインを活用し、自由に変更可能なページング機能を実装しました。

## 使用スキル
* CSS
* WordPress / PHP
* プラグイン：My Custom Post Types、Custom Upload Mimes、Custom Taxonomy、Advanced Custom Fields、WP-PageNavi
